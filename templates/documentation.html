Netflix API - Documentation
Configuration

GET /api/configuration/countries Return JSON with list of countries. https://flask-netflix-api.herokuapp.com/api/configuration/countries

GET /api/configuration/languages Return JSON with list of languages. https://flask-netflix-api.herokuapp.com/api/configuration/languages

GET /api/configuration/timezones Return JSON with list of timezones. https://flask-netflix-api.herokuapp.com/api/configuration/timezones

Credit

GET /api/type_media/media_id/credits Return JSON with credit ID. https://flask-netflix-api.herokuapp.com/api/movie/17302/credits

GET /api/credit?credit_id=credit_id Return JSON with credit details. https://flask-netflix-api.herokuapp.com/api/credit?credit_id=52fe47169251416c7508d5c7

Discover

GET /api/discover/type_media Return JSON movie or tv show to discover. https://flask-netflix-api.herokuapp.com/api/discover/movie?language=en-US&year=2001&with_genres=28&sort_by=popularity.asc

optional : accepted_sort = ['popularity.asc', 'popularity.desc', 'release_date.asc', 'release_date.desc', 'original_title.asc', 'original_title.desc', 'vote_average.asc', 'vote_average.desc']

optional : year

optional : language

optional : with_genres

Find

GET /api/search/type_media?query=Alice Return JSON list of TV shows / movies ID. https://flask-netflix-api.herokuapp.com/api/search/movie?query=Alice

GET /api/search/type_media/media_id Return JSON with TV show / movie details. https://flask-netflix-api.herokuapp.com/api/search/movie/17302

Genre

GET /api/genre/type_media/list Return JSON with list of genres for TV Shows / movies. https://flask-netflix-api.herokuapp.com/api/genre/movie/list

Media

GET /api/type_media/media_id/external_ids Return JSON with external IDs & social. https://flask-netflix-api.herokuapp.com/api/movie/17302/external_ids

Trending

GET /api/trending/type/time_window Return JSON with list of trending TV Shows / movies. https://flask-netflix-api.herokuapp.com/api/trending/movie/week

type: all, movie, tv

time_window: day, week
